<h3 className="font-bold text-xl text-gray-500">1) CI/CD for Laravel</h3>
<br/>
<ul className="list-disc pl-8">
  <li>Composer/npm caching; split tests (PHPUnit/Pest)</li>
  <li>Build once; promote artifact across envs</li>
  <li>Preview environments for risky PRs</li>
</ul>
<br/>

```yaml
# .github/workflows/laravel-ci.yml (excerpt)
name: CI
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    services:
      mysql:
        image: mysql:8
        env: { MYSQL_ROOT_PASSWORD: root, MYSQL_DATABASE: app_test }
        ports: ["3306:3306"]
        options: >-
          --health-cmd="mysqladmin ping -h 127.0.0.1 -proot"
          --health-interval=10s --health-timeout=5s --health-retries=5
    steps:
      - uses: actions/checkout@v4
      - uses: shivammathur/setup-php@v2
        with: { php-version: '8.3', coverage: none, tools: composer:v2 }
      - name: Cache Composer
        uses: actions/cache@v4
        with: { path: ~/.composer/cache/files, key: ${{ runner.os }}-composer-${{ hashFiles('**/composer.lock') }} }
      - run: composer install --no-interaction --prefer-dist
      - run: cp .env.ci .env && php artisan key:generate
      - run: php artisan migrate --force
      - run: vendor/bin/pest --ci || vendor/bin/phpunit --testdox
```

<h3 className="font-bold text-xl text-gray-500">2) Performance &amp; Caching</h3>
<br/>
<ul className="list-disc pl-8">
  <li>OPcache tuned; <code>config:cache</code> and <code>route:cache</code> in builds</li>
  <li>Redis for cache/sessions; Horizon to watch queues</li>
  <li>HTTP caching with CDN; image/WebP and font hygiene</li>
</ul>
<br/>

<h3 className="font-bold text-xl text-gray-500">3) Database &amp; Queues</h3>
<br/>
<ul className="list-disc pl-8">
  <li>Indexes for hot queries; fix N+1s with eager loading</li>
  <li>Queue isolation for critical work; idempotent jobs with retries/backoff</li>
  <li>Slow query sampling in APM; read replicas only where safe</li>
</ul>
